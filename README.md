# ItemDecoration

* __有以下几种分割线形式__


  > 1.横向分割线

  > 2.竖向分割线

  > 3.时间轴分割线

  > 4.带Header分割线

  > 5.粘性头部分割线


* __新增kotlin版本，涉及一下知识点__

  - `??`,`!!`,`?.`使用
  - `主构造函数`,`初始化代码块`,`init{}`等概念和代码结构执行顺序
  - `set`和`get`方法与`变量关系`
  - `by lazy`和`lateinit`的关系区别
  - `internal`和`inner`等关键字作用范围
  - `let`和`Elvis运算符`使用
  - `Map`的`put`和`get`方法
